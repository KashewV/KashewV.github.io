Monte Carlo FX Simulator

A tool for modeling currency risk using correlated random walks, Value at Risk (VaR), and scenario analysis.

View on GitHub

Project Overview

This simulator models the USD value of future revenues denominated in multiple currencies (EUR and INR). It uses a Geometric Brownian Motion model to forecast correlated FX rate paths, allowing for a comprehensive analysis of potential outcomes. The tool is designed for assessing risk, featuring metrics like Value at Risk (VaR) and Conditional Value at Risk (CVaR). The interactive web application is built with Streamlit, providing an intuitive interface for adjusting parameters and visualizing the results.

Key Features

  • Correlated Asset Paths: Simulates EURUSD and INRUSD price paths using a Cholesky decomposition to accurately model market correlations.
  • Flexible Cashflow Modeling: Allows for seasonal revenue patterns (even, front-loaded, back-loaded) to reflect real-world business cycles.
  • Forward Hedging Strategy: Users can specify partial forward hedging ratios for each currency to model and assess the impact of risk mitigation strategies.
  • Comprehensive Risk Metrics: Calculates and displays key risk indicators including 95% VaR, 95% CVaR, and the probability of loss relative to the base case.
  • Interactive Scenario Analysis: All major inputs, such as spot prices, volatilities, market drift, and interest rates, are fully adjustable through the Streamlit interface.

Technology Stack

  • Backend: Python, NumPy, Pandas
  • Web Framework: Streamlit
  • Plotting: Altair, Matplotlib
  • CLI: Typer
